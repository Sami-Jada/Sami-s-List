# Shared Types - Sami's List

Shared TypeScript types, interfaces, and enums used across backend and mobile applications.

## Usage

Import types from the shared package:

```typescript
import { User, Order, OrderStatus, UserRole } from '@samis-list/shared';
```

## Structure

- `types/index.ts`: All shared types, interfaces, and enums

## Types Included

- User types and roles
- Order types and statuses
- Payment types and methods
- Notification types
- Vendor and Driver types
- API response types
- Common utility types
